# Социальная сеть
**Реализованный функционал:**  
• Добавление/удаление друзей (с отправкой запроса)  
• Чат  
• Добавление/удаление постов  
• Поиск других пользователей  
• Авторизация  
• Смена аватара 

# Стек
**Frontend:**  
• Vue.js 3  
• Pinia  
• Vue-router  
• Vuetify  
• Axios

**Backend:**  
• Node.js (express)  
• Express-validator  
• ws  
• jsonwebtoken  
• bcrypt  
• pg-promise  

База данных - PostgreSQL
